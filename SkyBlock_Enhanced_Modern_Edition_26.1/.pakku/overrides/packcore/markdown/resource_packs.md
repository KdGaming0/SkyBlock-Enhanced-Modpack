# Resource Packs

Packs stack. The pack highest in the list wins when two change the same texture; the ones below fill in whatever's left. So lead with your main pack and stack addstions underneath.

Click a pack to select it — you can reorder them later in **Options → Resource Packs.**

---

## Pack Descriptions

### Hypixel Plus

[Hypixel Plus Preview](https://modrinth.com/resourcepack/hypixel-plus/gallery)

A clean, mostly-vanilla UI and item pack built for Hypixel — sharp icons, polished menus, and excellent readability without heavy restyling.

> **Needs a JVM flag:** `-Xss4M`. Finish setup, add the flag, then restart Minecraft before enabling this pack. Stuck? [Join the Discord.](https://discord.gg/pdwxyjTta7)

### FurSky Reborn

[FurSky Reborn Preview](https://modrinth.com/resourcepack/furfsky-reborn/gallery)

One of the most popular SkyBlock packs, now bundled into a single pack with built-in options. Custom items, GUIs, and full visual overhauls — and you can toggle features to taste, like items-only or the full experience.

> Don't stack the full GUI features with another UI pack like SkyBlock Dark UI; they'll conflict.

### SkyBlock Dark UI

[SkyBlock Dark UI Preview](https://modrinth.com/resourcepack/skyblock-dark-ui/gallery)

A dark theme for SkyBlock menus and mod interfaces. Smooth, modern, and easy on the eyes.

### Defrosted

[Defrosted Preview](https://modrinth.com/resourcepack/defrosted_pack/gallery)

A frost-blue 16× pack with a clean, minimalist style. Also comes in pink.

### Looshy

[Looshy Preview](https://modrinth.com/resourcepack/looshy/gallery)

A smooth vanilla-style 16× pack. Minimal and clear, keeping the Minecraft feel intact.

### Dark Mode SkyBlock

[Dark Mode SkyBlock Preview](https://modrinth.com/resourcepack/dark-mode-skyblock/gallery)

Darkens world zones like The End, The Mist, the Rift, and Glacite Tunnels — easy on the eyes during long sessions.

### PacksHQ

[PacksHQ Preview](https://modrinth.com/resourcepack/packshq/gallery)

One of the oldest Hypixel SkyBlock packs, focused on a strong vanilla feel with better clarity. Adds custom items, armor, ores, pets, GUIs, and more while staying true to the original style.

## Popular Combinations

- Hypixel Plus (solo)
- FurSky Reborn
- SkyBlock Dark UI + FurSky
- SkyBlock Dark UI + Hypixel Plus
- Defrosted + Looshy
- Dark Mode SkyBlock + any main pack
