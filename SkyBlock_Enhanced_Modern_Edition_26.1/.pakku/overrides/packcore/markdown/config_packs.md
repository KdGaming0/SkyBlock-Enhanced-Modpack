# Config Packs

SkyBlock Enhanced comes **pre-configured for your screen.** Overlays shift around as resolution changes, so each one gets its own tuned config — ready-made for **1080p, 1440p, 1440p Ultrawide, 4K, and 4K Ultrawide.**

On first launch the pack checks your resolution and applies the closest match automatically. The active pack is shown on the right; a green checkmark means it applied cleanly.

**Got the wrong one?** Pick another from the list on the right. On 1080p you can also switch the GUI scale to 2× instead of the default 3× if you'd like a bit more screen space.
