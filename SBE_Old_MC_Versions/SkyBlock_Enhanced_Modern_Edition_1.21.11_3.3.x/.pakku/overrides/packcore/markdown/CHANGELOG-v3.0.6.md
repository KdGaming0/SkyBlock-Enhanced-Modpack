# 🛠 Update 3.0.6
  
### Fix
- Freeze when launching the game on macOS        
- Tooltip scroll not working with modern UI custom tooltip
  
### Improvements
- The game now makes a config backup every time you update the pack. So if your config reset after an update of `/packcore modpack_configs`, go to the backup page, find the backup for the relevant version, and click restore.
  
## 🔄 Updated
### Mods
- **libjf**: 3.19.10 -> **3.19.11**
- **packcore**: 4.1.7+1.21.11 -> **4.2.1+1.21.11**
- **skyblock_enhancements**: 0.8.6+1.21.11 -> **0.9.0+1.21.11**
- **SkyHanni**: 7.12.0-mc1.21.11 -> **7.13.0-mc1.21.11**
- **waveycapes-fabric**: 1.8.2-mc1.21.11 -> **1.9.0-mc1.21.11**
  
## Removed
### Mods
- **tooltip-scroll** (Replaced by Skyblock Enhancements)
  
## Default configs (3.0.2)
- Inter font is now off by default, and users have to select it in the welcome wizard or with `/packcore modernui font inter` to use it. Vanilla font is now the default again.
  
---
  
### :tools: Troubleshooting & Tips
- The first launch after updating may take slightly longer than usual.
- If Minecraft appears frozen while loading, wait a moment before closing it. 
- If Modrinth does not show the update, refresh the instance page.
