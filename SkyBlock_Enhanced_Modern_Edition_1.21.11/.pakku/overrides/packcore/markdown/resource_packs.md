# Resource Packs

Packs can stack — the pack listed highest wins when two packs change the same texture. Packs lower down only fill in what's missing.

Click a pack to select it. You can change the load order later in **Options → Resource Packs**.

---

## Pack Descriptions

### Hypixel Plus

[Hypixel Plus Preview](https://modrinth.com/resourcepack/hypixel-plus/gallery)

A clean, mostly-vanilla UI and item pack built for Hypixel. Clear icons, polished menus, and great readability without heavy style changes.

> **Requires a JVM flag:** `-Xss4M`. Finish setup, add the flag, then restart Minecraft before enabling this pack. Need help? [Join the Discord](https://discord.gg/pdwxyjTta7)

### FurfSky Reborn (Full)

[FurfSky Reborn Full Preview](https://modrinth.com/resourcepack/furfsky-reborn/gallery)

A very popular SkyBlock pack. Replaces almost everything including GUI textures.

> Do not use alongside SkyBlock Dark UI — the two can break some menus together.

### FurfSky Reborn (Overlay)

[FurfSky Reborn Overlay Preview](https://modrinth.com/resourcepack/furfsky-reborn/gallery)

The item-textures-only version of FurfSky. Only changes SkyBlock item textures, leaving the GUI untouched. Pairs well with dark UI packs.

### SkyBlock Dark UI

[SkyBlock Dark UI Preview](https://modrinth.com/resourcepack/skyblock-dark-ui/gallery)

A dark GUI theme for SkyBlock menus and mod interfaces. Smooth, modern, and easy on the eyes.

### Defrosted

[Defrosted Preview](https://modrinth.com/resourcepack/defrosted_pack/gallery)

A frost-blue themed 16× pack with a clean minimalist style. Also available in pink.

### Looshy

[Looshy Preview](https://modrinth.com/resourcepack/looshy/gallery)

Smooth vanilla-style 16× pack. Minimalist and clear without changing the Minecraft feel.

### Dark Mode SkyBlock

[Dark Mode SkyBlock Preview](https://modrinth.com/resourcepack/dark-mode-skyblock/gallery)

Darkens SkyBlock world zones like The End, The Mist, the Rift, and Glacite Tunnels. Great for long sessions.

### Sophie's Enchants

[Sophie's Enchants Preview](https://modrinth.com/resourcepack/sophies-enchants/gallery)

Gives every enchanted book a unique texture so you can identify enchants at a glance in your inventory or chests.

## Popular Combinations

- Hypixel Plus (solo)
- FurfSky Full (solo)
- FurfSky Overlay + SkyBlock Dark UI
- Hypixel Plus + SkyBlock Dark UI
- SkyBlock Dark UI + Defrosted + Looshy
- Defrosted + Looshy
- Dark Mode SkyBlock + any main pack
- Sophie's Enchants + any pack (overlays cleanly on top)
