# Modern UI

Modern UI is a rendering library that enhances several visual aspects of the game and includes some small optimization improvements. The options on the right let you enable or disable some of its main features.

---

## Custom Font

When enabled, the game uses **Inter** — a clean, modern sans-serif typeface — instead of the default Minecraft font.

> **Requires a restart** to take effect. Font files are only loaded once at startup.

If you prefer the classic Minecraft look, or if text appears too thin or hard to read on your display, turn this off.

---

## Fancy Tooltips

Replaces the standard vanilla tooltip boxes with styled, rounded tooltips. Features include:

- Rounded corners with a soft shadow
- Gradient border that adapts to the item's rarity color
- Centered item name with a decorative title break

Turn this off if you use a resource pack that already styles tooltips, or if you prefer the vanilla look.

---

## Startup Ding

Plays a short sound effect when the game finishes loading. A small quality-of-life feature — you will know when the game is ready without watching the screen.

Turn this off if you find it annoying, or if you are using a custom startup sound via a resource pack.

---

> Changes to **Fancy Tooltips** and **Startup Ding** take effect immediately after applying.
> Changes to **Custom Font** require a **full game restart**.

For deeper configuration, open the **Mods** menu, find **Modern UI** in the list, and click it. From there you can change fonts and adjust many additional options.

You can also configure Modern UI features at any time using the command `/packcore modernui`.
