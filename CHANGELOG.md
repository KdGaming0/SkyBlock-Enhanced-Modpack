# 1 Million Downloads — and v5.0 is here!

Hey everyone! The pack has officially crossed **1 million downloads**. I honestly can't thank you all enough — when I started this I never imagined it reaching even a fraction of this many people, and knowing that so many of you load it up and find it useful is the best reward I could ask for.

To celebrate, I'm dropping one of the biggest updates the pack has ever had: **version 5.0**. (Totally not just a coincidence that this lined up with the milestone, and was definitely planned) With 5.0 have I done close to a complete pass over the whole pack, from the mod list to performance to the default configs.

## What's new

- **Updated to Minecraft 26.1**
- **Reworked Item Recipe list & view** for better browsing
- **~40% better average FPS in the Hub** with smoother frame times
- **Brand-new default configs (v5)** — with massive improvements for a cleaner first-time experience
- **Revamped mod selection** for better stability and features

Everything below is split into sections — expand whichever ones you want to know more about.

## Performance: 4.0 vs 5.0

- Average FPS in the Hub improved by **~40%**
- Noticeably fewer frametime stutters for smoother gameplay
- Game startup is **~12% faster** (roughly 2–3 seconds quicker, varies by system)

## Updating

For most people, you can simply **update as normal** and everything will be fine.

A fresh install is worth considering if:

- You haven't made major config changes and want a clean start on the new defaults, or
- You're running into issues with your current install.

On startup you'll be asked whether to apply the new default configs or keep your own. **If you've tweaked your configs to your liking, I recommend keeping your own.**

> If you've added your own mods to the pack, remember to update them to **26.1**.

## Updating

For most people, you can simply **update as normal** and everything will be fine.

A fresh install is worth considering if:

- You haven't made major config changes and want a clean start on the new defaults, or
- You're running into issues with your current install.

On startup you'll be asked whether to apply the new default configs or keep your own. **If you've tweaked your configs to your liking, I recommend keeping your own.**

> If you've added your own mods to the pack, remember to update them to **26.1**.

# 🛠 Update 4.0.0-beta.1

> This is a **beta** release. **Issues** may occur. For the most stable experience, wait for a non-beta version.

Hey everyone!

Today I have a big update for you all. 8 new mods have been added to improve your experience on Hypixel.

In addition, Roughly Enough Items has been replaced by Reliable Recipe Viewer. This change was made because RRV runs significantly better compared to REI. From my testing, you can get up to a 50% FPS increase while the inventory and item viewer is open, along with a slightly faster startup time.

RRV is only responsible for the item list and viewing items. My mod, Skyblock Enhancements, handles adding items to the list. If you run into any issues with RRV, please report them so they can be fixed as soon as possible.

Alongside that change, two new performance mods have been added that can significantly improve FPS in situations with many HUD elements on screen, such as in the Garden. In testing, FPS more than doubled in some cases while standing still with over half the screen covered in SkyHanni HUDs.

Feesh and Secret Routes Mod have also been added based on user requests.

That’s it for this update. Thank you all for using the modpack and for almost reaching 800,000 downloads! A lot of time goes into maintaining this pack, so if you want to support it, consider donating on [Ko-fi ](https://ko-fi.com/kdgaming1)or using code **KD1** at checkout when buying Hytale.

For users who have issues with the new font not rendering correctly, this can be fixed by running `/packcore modernui font inter` for custom font and `/packcore modernui font vanilla` for the vanilla font. Restart the game after.

## New Modpack Features

> Previews of all the new menus can be found here: https://modrinth.com/modpack/skyblock-enhanced-modern-edition/gallery

### First-Launch Setup
- New Welcome Wizard for guided first-time setup
- Choose your default config preset on the very first page of the wizard
- Choose from three main menu styles: Modern, Modern Minimal, Minimal

- New wizard options for storage design, main menu style, and sword block toggle
- Updated resource pack picker during setup
- New and improved final review page showing per-setting apply status

### First-Launch Setup
- New Welcome Wizard for guided first-time setup
- Choose your default config preset on the very first page of the wizard
- Choose from three main menu styles: Modern, Modern Minimal, Minimal
- Performance profile selector now includes a performance shader option
- New wizard options for storage design, main menu style, and sword block toggle
- Updated resource pack picker during setup
- New and improved final review page showing per-setting apply status

### Config Pack System
- Improved automatic config pack detection and applying
- Tracks applied pack and version for safer updates — when a new config is available, only new files are applied (such as configs for newly added mods). Existing mod configs are left untouched to avoid overwriting any changes you've made.

### In-Game Config Manager
- New in-game config screen with tabs: Configuration, Export, Import, Backups
- Browse config pack contents before applying
- Apply individual files or an entire preset at once
- Export selected files as a reusable `.zip` config pack with metadata (name, version, author, description, target resolution, GUI scale)
- Import external config packs from the imports folder
- Create and restore backups from within the game

### Commands
- `/packcore wizard` — Reopen the Welcome Wizard
- `/packcore modpack_config` — Open the config manager
- `/packcore update check` — Check for updates
- `/packcore update reset` — Reset update state
- All options from the wizard can also be changed with commands - just do `/packcore` and it will show autocomplete suggestions
