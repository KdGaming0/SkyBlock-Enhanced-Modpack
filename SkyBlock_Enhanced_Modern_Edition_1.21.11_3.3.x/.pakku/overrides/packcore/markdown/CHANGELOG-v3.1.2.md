# 🛠 Update 3.1.2

Hey, Skyblock Enhancements has been updated, adding many requested features and improving the in-testing RRV integration. The big one is chat search (Ctrl+F) and bug fixes.
ModernUI has also been removed, as it was unstable and buggy. If you were a big fan of the fancy tooltip and font, I will look into alternatives, and it may come back later. This update should finally stabilize the 1.21.11 version, so many of the issues users have been facing should now be gone. Thanks for the patience. Going forward, I will mark more releases as beta when larger changes are added, so if you want a stable experience, you know not to update.

## 🔄 Updated
### Mods
- **Dark Skyblock v1.1.zip**: 1.1 -> **1.2**
- **skyblock_enhancements**: 0.10.2+1.21.11 -> **0.11.1+1.21.11**

### 🛠 Troubleshooting & Tips
- The first launch after updating may take slightly longer than usual.
- If Minecraft appears frozen while loading, wait a moment before closing it.
- If Modrinth does not show the update, refresh the instance page.
