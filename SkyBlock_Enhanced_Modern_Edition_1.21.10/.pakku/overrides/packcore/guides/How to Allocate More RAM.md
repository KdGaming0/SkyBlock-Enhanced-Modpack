{gold}**How to Allocate More RAM**{}  

Allocating the right amount of memory helps your game run smoothly without wasting system resources. Follow this guide to check your RAM and configure memory settings in your launcher.  



{gold}**How Much RAM Should I Allocate?**{}  

The amount of RAM you should allocate depends on your system and the mods you’re using.  

> {#275EF5}**Important:**{} The maximum you can allocate is ***half of your total system RAM***.  

{yellow}**Check Your System RAM**{}  

1. Open Task Manager (`Ctrl + Shift + Esc`)  
2. Go to the *Performance* tab  
3. Select *Memory*  
4. You’ll see your total RAM  

![Task Manager Memory](packcore:textures/lavender/images/task_manager_memory.png,fit)  



{yellow}**Recommended Allocation**{}  

> Use these recommendations to find the right balance for your setup:  

- {gold}**8GB System RAM**{}  
  - Light Setup: 3GB  
  - Heavy Mods/Modpacks: 4GB  

- {gold}**16GB System RAM**{}  
  - Light Setup: 4GB  
  - Heavy Mods/Modpacks: 6–8GB  

- {gold}**32GB+ System RAM**{}  
  - Light Setup: 4GB  
  - Heavy Mods/Modpacks: 6–10GB  

{#275EF5}**Note:**{} Allocating too much RAM can hurt performance. Don’t allocate more than you need.  



{gold}**Allocating RAM in Modrinth App**{}  

{yellow}**For All Instances (Recommended)**{}  

1. Click the *Settings* icon (bottom-left sidebar)  
![Modrinth Sidebar Settings](packcore:textures/lavender/images/modrinth_settings_sidebar.png,fit)  
2. Select *Default instance options*  
3. Scroll to *Memory Allocated*  
![Modrinth Default Memory](packcore:textures/lavender/images/modrinth_default_memory.png,fit)  
4. Move the slider to your desired value  
    - 3GB = 3072MB  
    - 4GB = 4096MB  
    - 6GB = 6144MB  
    - 8GB = 8192MB  
5. Close the window  

{yellow}**For a Specific Instance**{}  

1. Open your instance  
2. Click the *Settings* icon (top-right, next to Play button)  
![Modrinth Instance Settings](packcore:textures/lavender/images/modrinth_instance_settings.png,fit)  
3. Select *Java and memory*  
4. Scroll to *Memory Allocated*  
5. Enable *Custom Memory Allocation*  
6. Move the slider to your desired value  
![Modrinth Instance Memory Slider](packcore:textures/lavender/images/modrinth_instance_memory_slider.png,fit)  
7. Close the window  



{gold}**Allocating RAM in Prism Launcher**{}  

{yellow}**For All Instances (Recommended)**{}  

1. Click the *Settings* button  
![Prism Settings Button](packcore:textures/lavender/images/prism_settings_button.png,fit)  
2. Select *Java* from the list  
3. Find *Maximum memory allocation*  
![Prism Default Memory](packcore:textures/lavender/images/prism_default_memory.png,fit)
4. Set it to your desired value  
    - 3GB = 3072MB  
    - 4GB = 4096MB  
    - 6GB = 6144MB  
    - 8GB = 8192MB  
5. Close the window  

{yellow}**For a Specific Instance**{}  

1. Select your instance  
2. Click *Edit* (right sidebar)  
![Prism Edit Instance](packcore:textures/lavender/images/prism_edit_instance.png,fit)  
3. Click *Settings* (right sidebar)  
4. Select *Memory*  
5. Find *Maximum memory allocation*  
6. Set it to your desired value  
![Prism Instance Memory](packcore:textures/lavender/images/prism_instance_memory.png,fit)  
7. Close the window  



{gold}**How to Tell If You Need More RAM**{}  

Press ***F3*** in-game and look at the top-right corner. If memory usage frequently reaches **90–95%**, you should allocate more RAM.  

![F3 Memory Usage](packcore:textures/lavender/images/f3_memory_usage.png,fit)
