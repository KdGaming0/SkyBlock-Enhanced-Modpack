## 🛠️ Building From Source

This modpack uses **Pakku** for version control and building. Follow these steps to build it yourself:

### Prerequisites

- Install Pakku: [Installation Guide](https://juraj-hrivnak.github.io/Pakku/installing-pakku.html)
- Git (to clone the repository)

### Build Steps

1. **Clone or download this repository**
```bash
    git clone https://github.com/yourusername/skyblock-enhanced-modern-edition.git
    cd skyblock-enhanced-modern-edition
```

2. **Choose your Minecraft version**

    - Navigate to the `1.21.11` folder

3. **Fetch the mods**
```bash
    pakku fetch
```

4. **Export the modpack**
```bash
    pakku export
```

5. **Find your build**

    - The exported modpack will be in the `build` folder under `modrinth`

### Updating the Modpack

**To update the version number:**

1. Edit `.pakku/overrides/packcore/modpack_info.json`
2. Edit `pakku.json` to match the new version

**To update all mods:**
```bash
pakku update --all
```

**Learn more:** [Pakku Documentation](https://juraj-hrivnak.github.io/Pakku/)
